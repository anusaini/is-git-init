# isGitInit

Check if git has been initialized in a repo.

## Install

```bash

    npm i -D is-git-init

```

## Usage

```javascript

    const isGitInit = require('@anu/is-git-init');
    isGitInit();

```

## note

Use `node-path-choice` package to set process.env.NODE_PATH in `index.js` and `Gruntfile.js`.
